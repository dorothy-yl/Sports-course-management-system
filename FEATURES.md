# 系统特性说明

## 🎯 核心特性

### 1. 响应式主布局

#### 桌面端 (≥ 768px)
- 侧边栏展开显示，宽度 200px
- 完整的导航菜单和文字标签
- 顶部导航栏显示用户昵称
- 表格和表单使用多列布局

#### 移动端 (< 768px)
- 侧边栏自动折叠，宽度 64px
- 仅显示图标，鼠标悬停显示文字
- 顶部导航栏隐藏用户昵称
- 表格和表单自适应为单列布局

#### 特点
- 平滑过渡动画（0.3s）
- 手动切换侧边栏状态
- 窗口大小变化自动适配
- 使用 Pinia 管理侧边栏状态

### 2. 分段视频编辑器

这是系统最核心和复杂的功能模块。

#### 功能特性
- **动态分段管理**
  - 添加/删除分段
  - 拖拽排序（未实现）
  - 批量操作（未实现）

- **智能参数显示**
  - 跑步机课程：速度 + 坡度
  - 脚踏车课程：速度 + 阻力
  - 划船机课程：仅阻力

- **实时验证**
  - 结束时间必须大于开始时间
  - 检测时间段重叠
  - 自动计算分段时长
  - 自动计算总时长

- **用户友好**
  - 卡片式布局，清晰直观
  - 每个分段独立编辑
  - 实时显示时长
  - 错误提示明确

#### 数据结构
```javascript
segment: {
  id: string,           // 唯一标识
  startTime: number,    // 开始时间（秒）
  endTime: number,      // 结束时间（秒）
  speed?: number,       // 速度（km/h）
  incline?: number,     // 坡度（%）
  resistance?: number   // 阻力（级别）
}
```

#### 管理功能

- 查看详情

- VIP 状态实时切换
- 删除用户（含确认）

## 📱 响应式特性

### 断点定义
```css
xs: < 768px   (手机)
sm: ≥ 768px   (平板竖屏)
md: ≥ 992px   (平板横屏/小桌面)
lg: ≥ 1200px  (桌面)
xl: ≥ 1920px  (大屏)
```

### 适配策略
- **布局**
  - 桌面：多列网格
  - 移动：单列堆叠

- **导航**
  - 桌面：展开菜单
  - 移动：折叠图标

- **表格**
  - 桌面：固定列
  - 移动：横向滚动

- **表单**
  - 桌面：双列布局
  - 移动：单列布局
